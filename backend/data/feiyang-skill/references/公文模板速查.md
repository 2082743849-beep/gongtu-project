# 公文模板速查

> 飞扬强调：公文不是看文种决定结构，而是先看题目要求中的"写作目的"。

## 17种公文写作目的与结构速查

| 文种/场景 | 写作目的（侧重点） | 主体结构 | 格式要求 |
|-----------|-------------------|----------|----------|
| 讲话稿：经验分享 | 介绍做法 | 一做法一效果 | 标题+称呼+正文 |
| 讲话稿：动员 | 提高认识+具体部署 | 背景/问题→做法/对策 | 标题+称呼+正文 |
| 讲话稿：总结表彰 | 表扬+总结+提要求 | 总结成绩→指出问题→部署下一步 | 标题+称呼+正文 |
| 讲话稿：开幕式 | 表示欢迎+介绍情况+祝愿 | 欢迎/感谢→情况介绍/意义→祝愿 | 标题+称呼+正文 |
| 调研报告 | 汇报发现+分析+建议 | 基本情况→主要问题→原因分析→对策建议 | 不用格式（提纲/要点/概要） |
| 实施意见 | 指导怎么做，可操作性强 | 总体要求→重点工作→保障措施 | 标题+正文 |
| 工作方案 | 安排具体工作 | 总体要求/背景意义→重点工作任务→保障措施 | 标题+正文 |
| 倡议书 | 呼吁行动 | 背景/问题→具体倡议→呼吁号召 | 标题+称呼+正文+落款 |
| 感谢信 | 感谢+表扬贡献 | 表达感谢→对方贡献/事迹→再次感谢/祝愿 | 标题+称呼+正文+落款 |
| 推荐材料 | 推荐理由+事迹 | 推荐对象简述→主要事迹→推荐理由总结 | 标题+正文+落款 |
| 编者按 | 点明意义、总结概况 | 背景→主要内容→意义/推荐 | 居中【编者按】+正文 |
| 短评 | 亮明观点+论证 | 事件概述→观点（赞/批/中立）→理由→总结 | 标题+正文 |
| 汇报材料 | 客观陈述+建议 | 背景/原因→做法/进展→成效/不足→下步安排 | 标题+正文 |
| 简报 | 快速传递信息 | 面上情况→典型做法/亮点→不足/建议 | 不用格式（提纲/要点/概要） |
| 宣传稿 | 展示做法/成效 | 做法→成效（做法前效果后） | 标题+正文 |
| 回帖 | 回应+表态 | 感谢关注+回应问题+表明态度/措施 | 网络回帖格式 |
| 讲解稿 | 引导参观/介绍 | 欢迎/引入→分区域/分主题介绍→总结/展望 | 讲解员身份+称呼 |

## 常见错误

| 错误 | 解决 |
|------|------|
| 按文种记模板而不是按写作目的判断结构 | 核心是写作目的 |
| 分不清"提纲"和"要点" | 提纲=框架（条目/序号），要点=内容（短句） |
| 讲话稿分不清写给领导还是群众 | 给领导→成绩+部署；给群众→感谢+动员 |
| 把分析理解当"评析现象"写 | 分析理解=题干给话你解释；观点题=题干给现象你表态 |
| 概括题分条不用序号 | 必须标123，不整合 |
