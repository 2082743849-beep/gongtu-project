# 决策启发式（9条）

### 1. 字数决定策略
"如果字数紧张→压缩。如果字数充裕→展开。字数是一切取舍的第一变量。"

### 2. 背景段果断跳读
"描述性语言、描述画面，果断跳过去。段落开头'随着……'是背景，首尾句是结论——中间描述全跳。"

### 3. 主体按自然段选点
"一般来说是一个自然段选一个点。概括题找点就是这样——一个自然段一个核心信息。"

### 4. 写完不看（考生适用，AI 反向执行）
"写完以后不要看，看完更闹心，影响后面答题。要看在动笔之前看。"
> AI 执行注：输出答案分析前自我检查——题型判断正确、要点来自材料、字数合理。

### 5. 不编造
"材料里有就写，材料里没有就不要编。名言不能现场编，对策在材料里没有就不要硬写。"

### 6. 不分段不整合
"概括题不分段，标123列要点。不要轻易整合，不要做感动自己的事情。"

### 7. 上位词替代
"建钢厂→建企业，路灯排水渠停车场→基础设施。概括经验就是找上位词。"

### 8. 对策题1:3比例
"问题和对策的字数比例大概1:3。如果要求一一对应，就一问题一对策、二问题二对策。"

**对策从哪里来**：

| 来源 | 操作 | 类型 |
|------|------|------|
| 1. 问题反推 | 材料说了什么问题，反着写就是对策 | 飞扬原生 |
| 2. 做法借鉴 | 材料里别人的好做法/建议，换个主语提炼 | 飞扬原生 |
| 3. 原因推导 | 材料分析了原因，针对原因提对策 | 推断扩展 |
| 4. 常识补充 | 宣传教育/完善制度/加强监管/加大投入 | 飞扬原生（教材模板） |
| 5. 政策背景 | 乡村振兴/高质量发展/基层治理等政策热词 | 推断扩展 |

### 9. 大作文开头不背模板
"开头不能背，尤其是不能背万能开头。开头没扣题三五秒就pass了。"
