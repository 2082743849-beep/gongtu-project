# 心智模型速查（7个）

## 模型1：数格子——以格为核心的全流程管理

**一句话**：从审题开始就以作答格数反推一切——写多少点、每点多少字、什么该写什么该砍。

**来源**：
- 150~200字→20分钟，300字→25分钟，400字→30分钟，500字→35分钟
- 大作文1000字精确分配：开头150字(6行)、分论点各250字(3个750)、结尾100字(3-4行)、标题25字

**应用**：分析题目时先关注字数，再反推作答策略。字数少→只写核心要点；字数多→展开。

---

## 模型2：题干定题型——问什么答什么

**一句话**：题型判断标准就是题干问法本身，不靠猜。

**判据**：
- 概括题：题干要求"概括做法/原因/变化/特点/作用/问题"
- 观点题：题干出现"看法、评价、评析、认识"
- 大作文："题干确定结构，材料确定分论点"

### 题干解码：看不懂时的破题路径（5步）

| 步骤 | 操作 |
|------|------|
| 1. 判断有无具体话题 | 有→分析型；无→内涵型 |
| 2. 判断题干类型 | 分析/理解/启示/看法→分析理解题 |
| 3. 虚词替换 | 抽象词→读材料首尾段找高频实词 |
| 4. 好事坏事判断 | 好事→找做法和好处；坏事→找问题和危害 |
| 5. 套用题型框架 | 应用文→套模板；对策题→反推+提炼 |

### 题型混淆快速判据

| 混淆对 | 判据 |
|--------|------|
| 概括做法 vs 概括作用 | 做法问"做了什么"，作用问"做得怎么样" |
| 分析理解 vs 观点题 | 题干给话让理解→分析理解；给现象让评价→观点题 |
| 对策题 vs 概括做法+建议 | 材料以问题为主→对策题；以做法为主→先概括再补建议 |

---

## 模型3：三级压缩法——按字数分层取舍

**一句话**：按三级递进压缩：主体可省→目的/凭借可压→二级做法可砍。

**压缩优先级**：
1. 先砍二级做法和举例
2. 再砍目的状语和凭借依托
3. 最后砍主体

---

## 模型4：框架先行——每题型一个固定作答骨架

**一句话**：考场上不需要"构思"，只需要"套框架+填要点"。

**大作文题干三种展开方式**：

| 展开方式 | 判据 | 操作 |
|----------|------|------|
| 内涵展开 | 题干虚/空泛 | 从材料找维度分开写，化虚为实 |
| 意义展开 | 话题具体+强调重要性 | 分析为什么重要 |
| 措施展开 | 话题具体+强调做工作 | 写怎么做好 |

**内涵型唯一判断标准**：题干有没有具体话题（经济/文化/环保/民生/基层治理/乡村振兴等）。没有→内涵展开。

### 大作文完整立意5步链路

| 步骤 | 操作 | 关键动作 |
|------|------|----------|
| 1. 题干解码 | 看题干有无具体话题 | 有→意义/措施展开；无→内涵展开 |
| 2. 定主题词 | 题干或材料首尾段提取 | 虚词→材料找高频实词替换 |
| 3. 定论述方向 | 题干或材料总结段提取 | 时代背景、政策方向、问题导向 |
| 4. 定总论点 | 材料首尾段+总结材料 | 主题词+论述方向→一句话总论点 |
| 5. 抽分论点 | 从其他各则材料提取 | 每则材料一个维度，避免交叉 |

> 飞扬强调"立意5分钟就能完成，难的是怎么写"。

### 五步审题法（公文题）

1. 先看写作目的定主体思路
2. 确定基本结构
3. 分配开头/主体/结尾字数比例
4. 根据题目要求体现语言风格
5. 看是否需要格式（提纲/要点/概要不用格式）

**字数黄金比例**：
- 400字：开头25~50字，结尾25字，剩余主体
- 500字：开头50~75字，结尾25~50字，剩余主体

## 模型5：做法-效果二分法

**一句话**：申论材料底层只有两种信息——做法和效果。"做法在前，效果在后"。

**应用**：
- 概括做法=只写做法不写效果
- 概括作用=以效果为主+少量做法
- 讲话稿经验分享=做法加效果的方式

**效果动词标志**：增强、提高、改善……能体现比较的动词后面跟的是效果。

---

## 模型6：方法论仲裁原则

**触发条件**：同一问题有两种以上方法论路径可选时，按以下顺序：
1. 飞扬在课堂上有没有明确说过？（查心智模型和启发式）
2. 如果没说，优先选考场上执行成本最低的方案
3. 如果执行成本相当，优先选飞扬在类似场景中的做法

---

## 模型7：三遍阅读法

- **第一遍（速读，1-2分钟）**：只读每段首尾句，画出段落大意
- **第二遍（精读，核心）**：逐段找点，做法/效果二分法标注
- **第三遍（查漏）**：快速扫一遍，四维检查（问题/原因/影响/对策）

> 飞扬坦诚"其实核心的就是一遍"，小题材料阅读不超过题目建议时间的40%。
