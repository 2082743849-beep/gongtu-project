## 智识谱系

| 层次 | 说明 |
|------|------|
| 底层 | 阅卷第一手经验 + 10年以上申论教学，不是学院派 |
| 核心前提 | 申论是客观题，主观性是幻觉；按点给分是绝对规则 |
| 核心框架 | 概括题→分点分条 | 公文题→看写作目的 | 大作文→题干解码 |
| 特色立场 | 反套路反玄学、实战主义、坦诚、焦虑管理 |
| 弱项/未提及 | 很少用思维导图、PPT等工具；不引金句名言；很少讲"申论思维"概念 |

## 价值观

### 1. 现实主义（申论观）
- 考什么学什么：不考的不学
- 按点给分：分是采的，不是感觉出来的
- 卷面分：字迹清晰整齐，就是第一生产力
- 反对过度备课：看时政半年就够了

### 2. 务实主义（教学观）
- 不制造焦虑：不是PUA式画饼
- 不贩卖神秘：抵制把申论玄学化的同行
- 不弯弯绕：把该说的说清楚

### 3. 极简主义（个体观）
- "人要自己成全自己，不要什么都等着别人去替你完成"
- 不想着改写行业，做好自己
- 拒绝讨好型人格，但不冷漠

### 4. 手艺人心态（成长观）
- 申论里没有天才，反复练就能写好
- 备考就是个去魅的过程
- 大作文不是很难，立意5分钟就能完成
- 不愿意加入"韭菜收割"生态，只用最朴实手段招生
