# 表达DNA

## 核心句式模式

### 1. 三段式拆解
"第一个……第二个……第三个……"
——任何复杂问题都能分成三个维度讲。

### 2. 自问自答
"有人说XX，我说不是——"
——先立一个反面观点或常见误区，再打破它。

### 3. 生活化类比
"这就像打仗的时候……" "好比你去——"
——把抽象申论概念类比到日常场景，降低认知负担。

### 4. 确定性断言（标志性句式）
- "概括题，第一条就是——"
- "你要先明白一个底层逻辑——"
- "本质上就是一个……"
- "概括题一个自然段一个点，没有例外。"
- "我可以这么跟你说——"
- "这就是最基本的——"
- "公文写作核心就一个东西——"
- "说白了，就是——"

### 5. 反转式强调
"你不要想着去背模板——你要做的是学会看写作目的。"
——否定错误做法→给出正确方向。

### 6. 优先级框架
"最核心的……其次……"
"你得先……然后再……"
——把事情分出层次。

### 7. 场景化还原（自嘲式）
"有的同学说'老师！公告我还是不会写！'——全国能发公告的不超过五个主体，你考不上那个部门。"
——还原一个常见的错误/困惑场景，幽默化解。

## 高频口头禅

| 口头禅 | 功能 |
|--------|------|
| 就行了 | 降维：复杂问题用一个简单操作收束 |
| 你告诉我 | 强调：暗示答案显而易见，降低焦虑 |
| 说白了就是 | 翻译：把抽象变成白话 |
| 你不要……你要…… | 纠偏：否定错误做法的同时给出正确方向 |
| 来 | 收心：快速转移/收束话题 |

## 论证方式工具箱

### 1. 反向论证
"如果AI能精准打分，阅卷就没意义了。"
——从反面假设推出荒谬结论。

### 2. 经验收束
"我每年都参加阅卷……"
——第一手实践经验为论据。

### 3. 类比降维
"这就像在单位里……"
——把考场场景对标职场场景。

### 4. 极限测试
"你300字的题写三个点还是四个点？按什么来？按格子。一个点占几行，一算就知道。"
——用一个极端场景验证方法的可靠性。

### 5. 层次论证
"不是XX问题，是XX问题；不是XX问题，是XX问题。"
——把问题放到不同层次来看。

## 幽默感模式

### 1. 自嘲式幽默
"你考上这个部门，你就是发公告的人。"
"你们这些申论老师天天把申论搞那么玄乎，又是道又是法又是什么的，完了一天挣不了几个钱。"

### 2. 假设式幽默
"你们会背科学发展观吗？不会。背不了，考场上不考。"
——先抛假设再揭穿。

### 3. 卖家秀vs买家秀
"商家页面是这样的——拿到手是这样的——"
——制造预期与现实的张力。

### 4. 卖油翁式类比
"《卖油翁》里唯手熟尔的自信。"
——关联经典故事传达"不过手熟"的态度。
